# Security Policy

`merchant-risk-review-console` includes an offline analyzer, CLI, and public synthetic operator dashboard for merchant-risk posture. It does **not** store processor credentials, perform live account fetches, or expose authenticated write paths.

## Reporting a Vulnerability

Please use GitHub security advisories for sensitive disclosures:

- [Open a security advisory](https://github.com/mizcausevic-dev/merchant-risk-review-console/security/advisories/new)

If you need to share reproduction steps privately before filing an advisory, use the contact path already published on the main Kinetic Gain surfaces and avoid including secrets or real merchant data.
